Funcionalidades
Gerenciamento de Eventos: Cadastro, edição e controle de eventos
Gerenciamento de Usuários: Sistema de usuários integrado
Interface Gráfica: Interface amigável desenvolvida em Java Swing/AWT
Persistência de Dados: Sistema de arquivos para armazenamento
Arquitetura MVC: Organização clara entre Model, View e Controller

Tecnologias Utilizadas
Java: Linguagem principal do projeto
Java Swing/AWT: Interface gráfica
Arquitetura MVC: Padrão de organização do código
Sistema de Arquivos: Persistência de dados
